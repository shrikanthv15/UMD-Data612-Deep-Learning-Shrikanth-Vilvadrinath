## CLASS PROJECT

Transformer neural network model is a novel architecture that aims to solve sequence-to-sequence tasks while handling long-range dependencies with ease.  
Diffusion models are a type of generative model, particularly used for image generation.  
In your class project with your team, you will implement either a transformer model to predict time sequences using time series data or generate images using a diffusion model.  
You can identify a problem space and obtain a sample dataset.  
Prepare a 2-3 page proposal document, present background, significance and statement of the problem, how to obtain the training data, the metric for validation and verification and the DL framework to implement the transformer or diffusion model.  
You are allowed to follow one of the existing implementations so long as you cite the resource and adapt it to your problem space.
